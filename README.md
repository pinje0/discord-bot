#### Note

Guild = Server

Make the Bot online:

1. Running the code
2. Either using `node src/index.js` or `nodemon src/index.`
3. Or u can use script from `package.json`:

```json
"scripts": {
    "start": "node src/index.js",
    "dev": "nodemon src/index.js"
  },

```

Type in terminal:

```sh
npm run start
```

or

```sh
npm run dev
```
